Let $X$ and $Y$ be sets and let $f:X\to Y$ be a function. Let $S\subseteq Y$. Then the set 
$$
f^{-1}(S) = \{ x\in  X: f(x) \in  S \}
$$
is called the **inverse image** of the set $S$ under the function $f$


It is always the case that
$$\begin{align}
x\in   f^{-1}( f(x) ) \\
S\subseteq f^{-1}(f(S))
\end{align}
$$
even if the inverse function $f^{-1}(x)$ itself does not exist



# Examples

$f:\mathbb{R}\to \mathbb{R}$
$f(x)=x^{2}$ (inverse function doesn't exist, but inverse image does)
Find $f^{-1}(S)$

a) $S=(4,9)\to f^{-1}(S)=(-3,-2)\cup (2,3)$

b) $S=(-2,1)\to f^{-1}(S)=(-1,1)$

c) $S= \{ -5 \}\to f^{-1}(S)=\emptyset$

d) $S=\mathbb{R}^{+}\to f^{-1}(S)=\mathbb{R}\setminus \{ 0 \}$
